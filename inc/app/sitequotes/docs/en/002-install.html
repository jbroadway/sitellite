SITEQUOTES INSTALL INSTRUCTIONS

1. Install the necessary database tables:

$ mysql -p -u USER DBNAME < inc/app/sitequotes/install/install-mysql.sql
(enter password when prompted)

2. Copy the SiteQuotes collection definition into the cms app:

$ cd /PATH/TO/SITELLITE
$ cp -R inc/app/sitequotes/install/sitequotes_entry.php inc/app/cms/conf/collections/
